0x1D. C - Binary trees

This project involves solving various questions on binary trees as data structures. 
The problems solved are aimed at enhancing understanding of binary trees, tree traversal, inary search tree etc. 

The expected outcome is the ability to answer the folllowing questions.

What is a binary tree
What is the difference between a binary tree and a Binary Search Tree
What is the possible gain in terms of time complexity compared to linked lists
What are the depth, the height, the size of a binary tree
What are the different traversal methods to go through a binary tree
What is a complete, a full, a perfect, a balanced binary tree.

Tas 0: Creating a binary tree node

Task 1: Inserting node as the left child of another node

Task 2: Inserting a node as the right child of a given node

Task 3: Deleting an entire binary tree

Task 4: Check if a node is a leaf (has no children)

Task 5: Check if a given node is root (has no parent)

Task 6: Preorder traversal of a tree

Task 7: Inorder traversal of a tree

Task 8: Postorder traversal of a tree

Task 9: Measure the height of a tree

Task 10: Measure the depth of a node

Task 11: Measure the sixxe of a tree

Task 12: Count the number of leaves of a tree

Task 13: Count nodes with at least one child

Task 14: Measure the balance factor of a tree

Task 15: Check if a tree is full

Task 16: Check if a binary tree is perfect

Task 17: Find the siblings of a node

Task 18: Find the uncle of a node
